# Down4More — Python prototype (archived)

The Python + HTML prototype that originally shipped under this folder has been
moved to `python/legacy/`. It is preserved for reference but is no longer
maintained.

The current implementation lives at `Year 1/comSociety/Down4More/flutter/` and
ships as a native Flutter app for Linux / Windows / macOS / Android. The
Flutter app has reached feature parity with the Python prototype (and adds
download history, OS notifications, real OS-level pause/resume, metadata-aware
subtitles, per-item quality + format in playlist/batch, etc.).

If you want to run the legacy prototype anyway, see `legacy/README.md`.
