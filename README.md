# classes-projects

Coursework projects organized by year.

## Layout

```
.
├── Year 1/
│   └── comSociety/
│       └── Down4More/        # Multi-platform video downloader & slicer
├── Year 2/                   # placeholder for future coursework
└── Year 3/                   # placeholder for future coursework
```

Each year folder groups projects by course. To add a new project, create
`Year N/<course-name>/<project-name>/` and drop the project in there.
